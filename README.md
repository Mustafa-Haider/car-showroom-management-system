# Car Showroom Management System

## Developed By
Syed Mustafa Haider Rizvi, Axis Verma

## Overview
Car Showroom Management System is a Python-based project designed to simulate the operations of a car showroom by managing car inventory and employee records efficiently using binary file handling (`pickle` module).

This project allows users to store, retrieve, update, search, and delete records permanently through external binary files. It provides a menu-driven interface for smooth interaction and demonstrates practical implementation of file handling, modular programming, and CRUD operations in Python.

---

## Features

### Car Management
- Add new car records  
- Display all car records  
- Search car by ID  
- Update car details (price, availability)  
- Delete car records  

### Employee Management
- Add employee records  
- Display all employee records  
- Search employee by ID  
- Update employee details (salary, contact)  
- Delete employee records  

### Summary Reports
- Count cars by brand  
- Display employees with salary above a specified threshold
